Programming test

17 February - Afternoon

Phase 2, in the afternoon, is the programming phase and it will last for 3 hours. It is reserved for the participants who did the best job in the morning. The short list of students admitted to phase 2 will be announced in the early afternoon.

During the programming phase you will be asked to solve programming problems using the languages of your choice between C, C++, Python 2, Python 3 and Java (for information about laboratories tools please refers to the university of your choice at venues page).

By combining the results of the two phases we will define the final ranking: the top-20 students for each site will be selected to form the team for the edition 2020 of CyberChallenge.IT.

You can familiarize yourself with the new training and admission platform with the example problems.

You can train with programming tests of previous editions (2017, 2018 and 2019) or with example tests for the 2020 edition.

New example programming tests will be added later, stay tuned!
